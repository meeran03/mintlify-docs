---
title: 'Update Graph'
api: 'PUT /api/v1/assistant-graphs/{graph_id}'
description: 'Update an assistant graph metadata'
---

Update the metadata of an existing assistant graph. This endpoint updates only the graph-level properties like name, description, and configuration. To update nodes and edges, use the [Update Structure](/api-reference/assistant-graphs/structure) endpoint.

## Path Parameters

<ParamField path="graph_id" type="integer" required>
  The unique identifier of the graph to update.
</ParamField>

## Request Body

Only fields provided in the request will be updated.

<ParamField body="name" type="string">
  New name for the graph. Must be unique within your organization.
</ParamField>

<ParamField body="description" type="string">
  New description for the graph.
</ParamField>

<ParamField body="is_active" type="boolean">
  Whether the graph should be active. Set to `false` to disable the graph without deleting it.
</ParamField>

<ParamField body="configuration" type="object">
  Graph-level configuration settings.
</ParamField>

## Response

Returns the updated graph object with full details including nodes and edges.

<ResponseField name="id" type="integer">
  Unique identifier for the graph.
</ResponseField>

<ResponseField name="organization_id" type="integer">
  Organization that owns this graph.
</ResponseField>

<ResponseField name="name" type="string">
  Updated name of the graph.
</ResponseField>

<ResponseField name="description" type="string">
  Updated description of the graph.
</ResponseField>

<ResponseField name="is_active" type="boolean">
  Whether the graph is active.
</ResponseField>

<ResponseField name="nodes" type="array">
  Array of nodes (unchanged by this endpoint).
</ResponseField>

<ResponseField name="edges" type="array">
  Array of edges (unchanged by this endpoint).
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of the update.
</ResponseField>

<RequestExample>
```bash cURL
curl -X PUT https://api.burki.dev/api/v1/assistant-graphs/1 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated Customer Service Router",
    "description": "Routes customers to sales, support, or billing"
  }'
```

```bash cURL (deactivate graph)
curl -X PUT https://api.burki.dev/api/v1/assistant-graphs/1 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "is_active": false
  }'
```

```python Python
import requests

response = requests.put(
    "https://api.burki.dev/api/v1/assistant-graphs/1",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "name": "Updated Customer Service Router",
        "description": "Routes customers to sales, support, or billing"
    }
)

graph = response.json()
print(f"Updated graph: {graph['name']}")
```

```javascript JavaScript
const response = await fetch('https://api.burki.dev/api/v1/assistant-graphs/1', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Updated Customer Service Router',
    description: 'Routes customers to sales, support, or billing'
  })
});

const graph = await response.json();
console.log(`Updated graph: ${graph.name}`);
```

## Error Responses

| Status Code | Description |
|-------------|-------------|
| 401 | Unauthorized - Invalid or missing API key |
| 404 | Graph not found in your organization |
| 500 | Server error updating the graph |

<Tip>
To update the graph structure (nodes and edges), use the [Update Structure](/api-reference/assistant-graphs/structure) endpoint instead.
</Tip>
