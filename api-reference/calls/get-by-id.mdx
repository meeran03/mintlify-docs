---
title: 'Get Call by ID'
openapi: 'GET /api/v1/calls/{call_id}'
---

This endpoint retrieves a single call record by its unique database ID.

### Path Parameters

- `call_id` (integer, **required**): The unique identifier for the call you want to retrieve.

### Response

The response is a JSON object containing the full details of the specified call. The structure is the same as the objects returned by the [List Calls](/api-reference/calls/list) endpoint.
