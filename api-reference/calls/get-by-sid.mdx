---
title: 'Get Call by SID'
openapi: 'GET /api/v1/calls/sid/{call_sid}'
---

This endpoint retrieves a single call record by its unique Twilio Call SID. This is useful when you have the SID from Twilio's webhooks or logs and need to look up the corresponding call in your system.

### Path Parameters

- `call_sid` (string, **required**): The unique Call SID from Twilio.

### Response

The response is a JSON object containing the full details of the specified call. The structure is the same as the objects returned by the [List Calls](/api-reference/calls/list) endpoint.
